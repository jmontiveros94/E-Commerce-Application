# E-Commerce-Application
a back end for my e-commerce websiteAn E-Commerce site that uses the latest technologies that competes with other e-commerce companies

## Description

The E-Commerce Application is an online platform that enables users to browse, purchase, and manage products in a convenient and secure manner. It provides a seamless shopping experience with features such as product listings, user authentication, shopping cart functionality, and order management.

## Installation

1. Clone the repository
2. Navigate to the project directory
3. Install dependencies: 'npm i' 
4. Run 'npm seed'
